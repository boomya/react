### run getting-started sample
```sh
npm install --save react
npm install -g gulp
npm install --save-dev gulp gulp-browserify gulp-concat gulp-react gulp-connect lodash reactify
bower install bootstrap --save
gulp serve
```
http://localhost:5000/
